### Get Weather API Key (Optional)

1. Visit: https://openweathermap.org/api
2. Sign up (free)
3. Get API key
4. Add to `.env.local`:
   ```
   NEXT_PUBLIC_WEATHER_API_KEY=your_key_here
   ```
5. Restart server
